Get Started

Installation

Install pick-list via NPM or include it directly in your HTML. Follow the guide below to get started:

1. Install via NPM
npm install picklist

After installing, import or require the library in your project:

// Using ES6 import
import picklist from 'picklist';
// or using CommonJS require
const Picklist = require('picklist');
2. Include via <script> Tag

Alternatively, use a CDN to include the library in your HTML:

<!-- Include the dropdown-list library (replace with actual CDN link) -->
<link rel="stylesheet" href="https://cdn.example.com/dropdown-list/dist/dropdown-list.min.css">
<script src="https://cdn.example.com/dropdown-list/dist/dropdown-list.min.js"></script>

Ensure you include the library's CSS (if provided) for default styles, and place the script at the end of your body for proper loading.

Usage

Using the dropdown-list component is straightforward:

1. Import the Web Component
<script src="dropdownList.js"></script>
2. Use the custom element in your HTML
<dropdown-list placeholder="Select a fruit..."></dropdown-list>
3. Initialize with items in JavaScript
const dropdown = document.querySelector('dropdown-list');
dropdown.setItems([
  { value: 'Apple', keywords: ['fruit', 'red', 'sweet'] },
  { value: 'Banana', keywords: ['fruit', 'yellow', 'tropical'] },
  { value: 'Orange', keywords: ['fruit', 'citrus', 'vitamin c'] }
]);

API Reference

The dropdown-list library provides a simple API for customization and control. Below are some common methods and options:

  • new DropdownList(element, options): Initializes a dropdown on the given element (selector or DOM node). Options may include:
    • items: Array of items for the dropdown (if not using a select element).
    • search: Boolean to enable/disable search filtering (default true).
    • placeholder: Placeholder text when no item is selected.
    • multiple: Boolean to allow multiple selection (if supported).
  • dropdown.open(): Programmatically open the dropdown menu.
  • dropdown.close(): Programmatically close the dropdown menu.
  • dropdown.select(value): Select an item programmatically by value.
  • dropdown.clear(): Clear the current selection.

Refer to the official documentation for a full list of options and methods. The above covers the most commonly used API features for dropdown-list.

Events

The dropdown-list library emits custom events to help you hook into user interactions. You can listen for these events on the dropdown element:

  • dropdown-select: Fired when an option is selected. The event detail typically contains the selected value or item.
  • dropdown-input: Fired when the search input value changes (each time the user types). The event detail may contain the current input string.

Below is an example of attaching event listeners to capture these events:

<script>
  const dropdownElement = document.getElementById('myDropdown');
  // Listen for selection events
  dropdownElement.addEventListener('dropdown-select', function(event) {
    console.log('Selected item:', event.detail.value);
    // You can also update your UI or perform actions based on the selection
  });
  // Listen for input (search) events
  dropdownElement.addEventListener('dropdown-input', function(event) {
    console.log('Current search text:', event.detail.search);
    // This event can be used to implement dynamic suggestions or other interactions
  });
</script>

By leveraging these events, you can integrate custom logic whenever a user selects an item or types in the dropdown's search field. This makes the component highly extensible and adaptable to various use cases.
